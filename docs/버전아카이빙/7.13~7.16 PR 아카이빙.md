- **홈** — 공지사항(관리자 인라인 편집), 역량 추천 배너, 신규 강의
- **사용자화면 튜토리얼** 추가
- **나의 학습 대시보드** — 통합 학습 현황 · 월별 학습 추이 · 학습 알림

- **수강신청** — 바로 수강신청 → 관리자 승인 대기 플로우, 사내 개설 과정 신청(신청형/선발형)
- **개별학습** (사외교육·자격증·학위) — *사전신청·승인 절차 제거* → **등록 → 학습률 갱신 → 100% 달성 시 수료증 발급**(수료번호 자동 채번, 수료증 보기/인쇄·PDF)
- **역량 추천 강의** — 진단 결과 × 추천 설정(rec-config) 매칭 (문구: "약점 역량 추천 강의" → "역량 추천 강의")

### 관리자 콘솔 (6개 탭)

- **통합 학습 현황** — 라이트 테마 대시보드 재구성: 기준연도·기간·이름/아이디·부서 필터, KPI 카드(총 학습시간 추정·수강·수료·평균 진도율), 월별 학습 추이(스택 차트), 진도율 분포, 부서별/인기 강의 차트(SVG 자체 구현), 대상자 목록(페이지네이션 + 행 클릭 수강/수료 상세), 엑셀 내보내기
- **조직도·리마인드** — 조직도 데이터 소스를 **엑셀 "조직도 전체" 시트(364명) → `org-data.json`** 로 교체(Graph 폴백 유지), **대분야 ESTsoft/ESTsecurity 2개로 고정**, 학습 데이터는 이메일→사번 순 실시간 조인, 리마인드 발송·부서별 엑셀 다운로드 유지
- **그룹 멤버 관리** — 계정 등록/수정/비밀번호 재설정/삭제
- **수강신청 승인** — 대기/완료/불가 상태 관리, 반려 사유 기록
- **개별학습 이력관리** — 승인 처리 대신 학습률·수료증(번호/발급일) 조회 + 삭제 관리로 개편
- **추천강좌 관리 / 강좌 개설·운영** — 추천 풀·태그 가중치, 학습자원 등록·학사운영·교육이수 관리

### 서버 / 문서
