## v0.4 (7/20) — 검색·커뮤니티·승인 플로우·SSO·튜토리얼 고도화

### 1. 검색 서비스
- **검색 필터 바**: 난이도·평점·학습시간·수료증 유무·가격·인기 태그 조합 필터 + 초기화
- **추천 검색어**: 관리자가 등록(노출위치·기간·상태) → 헤더 검색창 제안·플레이스홀더·강의목록 칩으로 노출
- 관리자 콘솔 **검색서비스 관리** 패널 (동의어 사전은 제외)

### 2. 커뮤니티 · 질문&답변 (강의별 Q&A)
- 강의 상세 "커뮤니티" 탭을 실제 Q&A로 구현 (질문 등록·목록·답변)
- 관리자 답변은 **지식공유자 뱃지**로 강조, 본인/관리자만 삭제

### 3. 수강신청 승인 플로우
- 바로 수강신청 → **승인중** → 관리자 **승인/승인불가** → 사용자 알림 → 승인 시 **내 학습(진행중·목록)** 자동 반영
- 관리자 콘솔 **수강신청 승인** 패널 (대기/승인/불가 필터·사유)

### 4. 개별학습(사외교육·자격증·학위) 모듈
- **사전신청 → 관리자 승인 → 결과보고 → 이수 인정**의 4단계 상태 흐름
- 관리자 콘솔 **개별학습 이력관리** 패널 (유형·구분·승인여부 필터, 사전/사후 승인·반려)

### 5. Microsoft Teams(Entra ID) SSO
- OAuth2 **Authorization Code Flow**(무의존성 Node 구현), 조직도용 Azure 앱 재사용/전용 앱 분리
- 로그인 시 **이메일·이름 자동완성 회원가입**(이메일 고정, 서명 쿠키로 안전 전달)
- Azure 앱등록 권한 이슈 대비 **데모 모드**(실제 SSO 미설정 시 UI 흐름 시연) — `TEAMS_SSO=1`로 실제 전환

### 6. 온보딩 튜토리얼 개편
- 기획서(화면설명 수정본) **실제 화면 캡처**로 6단계 이미지·내용 전면 교체
  (홈 · 강의정보/내학습 · 수강신청 · 강의소개/커리큘럼 · 강의재생 · 역량진단·추천)
- **회원가입 필수화**(모든 항목 미입력 시 진행 불가, 건너뛰기로 우회 불가)
- 건너뛰기 동선 정리(로그인 시 역량진단으로)

### 7. 홈 추천 배너 · 역량진단 UI
- 초록 배경 제거 → **강의 썸네일 모자이크(광고 배너형)** 그리드, "추천강좌" 헤더 + "진단 다시 하기"
- 역량진단 페이지 **건너뛰기 → 홈**, 홈 빈 상태 진단 시작 CTA

### 8. 서버 안정화 (버그 수정)
- 인프런 API 일시 오류(`fetch failed`)가 대시보드/카탈로그 **캐시를 영구 오염**시키던 버그 수정
  (실패 시 `building` 초기화 → 다음 요청 재시도)
- `inflearn()` 호출에 **자동 재시도(최대 3회, 백오프)** 추가 → 대량 병렬 fetch 중 간헐 500 해결
